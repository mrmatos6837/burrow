<task type="auto">
  <name>Task 1: Refactor burrow-tools.cjs for --json flag, render wiring, and command cleanup</name>
  <files>.claude/burrow/burrow-tools.cjs</files>
  <action>
Refactor `burrow-tools.cjs` to add global --json flag handling and wire all command output through render functions when not in JSON mode.

**Global changes:**
1. Parse `--json` from `process.argv` before command switch: `const jsonMode = process.argv.includes('--json')`
2. Filter `--json` from argv before passing to command parsers to avoid ERR_PARSE_ARGS_UNKNOWN_OPTION
3. Require `./lib/render.cjs` at top of file
4. Update available commands in error messages: remove `list`, `children`; commands are now `add, edit, delete, move, get, dump, path, archive, unarchive`

**Per-command refactoring:**

`get` command:
- Add `--full` flag to parseArgs options: `full: { type: 'boolean', default: false }`
- If `jsonMode`: use existing renderTree() + core.output() path (unchanged)
- If not jsonMode:
  - Get card data: if id provided, use findById(); if no id, synthesize root card: `{id: '(root)', title: 'burrow', created: data.cards[0]?.created || new Date().toISOString(), archived: false, body: '', children: data.cards}`
  - Get breadcrumbs: if id provided, use getPath() and strip last element (the card itself), map to {id, title}; if no id, empty array
  - For depth > 1 or depth 0: pass children data from renderTree cards array filtered to the appropriate depth levels. The card object's children should reflect what renderTree returns for display.
  - Call `render.renderCard(card, breadcrumbs, { full: values.full, termWidth: process.stdout.columns || 80, archiveFilter })`
  - Write to stdout and exit 0

`add` command:
- If `jsonMode`: core.output(result) (unchanged)
- If not jsonMode:
  - After addCard(), call getPath(data, result.id) to get breadcrumbs
  - Strip last element and map to {id, title} for breadcrumbs
  - Call `render.renderMutation('add', result, { breadcrumbs, card: result, termWidth: process.stdout.columns || 80 })`

`edit` command:
- Capture old title and body BEFORE calling editCard(): `const oldTitle = card.title; const oldBody = card.body;` (find card first via findById)
- If `jsonMode`: core.output(result) (unchanged)
- If not jsonMode:
  - Get breadcrumbs via getPath()
  - Call `render.renderMutation('edit', result, { breadcrumbs, card: result, oldTitle, oldBody, termWidth: process.stdout.columns || 80 })`

`delete` command:
- If `jsonMode`: core.output(result) (unchanged)
- If not jsonMode:
  - Call `render.renderMutation('delete', result, {})` -- result already has {id, title, descendantCount}

`move` command:
- Capture source parent title BEFORE moving: find parent of card via findParent(), get parent title (or "root" if parent is null)
- If `jsonMode`: core.output(result) (unchanged)
- If not jsonMode:
  - Get target parent title: if newParentId is null, "root"; else findById and get title
  - Call `render.renderMutation('move', result, { fromParentTitle: sourceParentTitle, toParentTitle: targetParentTitle })`
  - Note: result from moveCard is the full card object; renderMutation needs result.id, result.title

`archive` / `unarchive` commands:
- If `jsonMode`: core.output(result) (unchanged)
- If not jsonMode:
  - Call `render.renderMutation('archive'|'unarchive', result, {})` -- result has {id, title, descendantCount}

`path` command:
- If `jsonMode`: core.output(cleanPath) (unchanged)
- If not jsonMode:
  - Call `render.renderPath(cleanPath)` where cleanPath is the [{id, title}] array

`dump` command:
- Keep as alias for `get --depth 0`. Same --json routing logic as `get`.

**Error handling:**
- If `jsonMode`: use core.errorOut() (unchanged -- outputs JSON error)
- If not jsonMode: write `render.renderError(message)` to stdout and exit 1
- Create a helper function `handleError(message, code, jsonMode)` to avoid duplication

**Remove `list` and `children` cases entirely from the switch statement.**

Output format: when not in JSON mode, write rendered string to stdout via `process.stdout.write(rendered + '\n')` then `process.exit(0)`.
