  <name>Task 1: renderTree, countActiveDescendants, archiveCard, unarchiveCard in mongoose.cjs</name>
  <files>.claude/burrow/lib/mongoose.cjs, .claude/burrow/test/mongoose.test.cjs</files>
  <behavior>
    - renderTree(data, rootId, {depth, archiveFilter}) returns {breadcrumbs, cards} flat render array
    - Each card in render array: {id, title, depth, descendantCount, hasBody, bodyPreview, created, archived}
    - depth default (no arg) = 1 (card + direct children); depth 0 = Infinity (full tree); depth N = N levels
    - breadcrumbs: null for root views; array of {id, title} ancestors (excluding target) for focused views
    - rootId=null/undefined: walks from root (data.cards), breadcrumbs null
    - rootId specified: includes the root card itself at depth 0, then its children at depth 1, etc.
    - archiveFilter: 'active' (default) excludes archived; 'include-archived' includes all; 'archived-only' shows only archived
    - countActiveDescendants(card) counts non-archived descendants recursively
    - descendantCount in render entries uses countActiveDescendants (except in archived-only mode)
    - Cards at depth cutoff boundary appear in array but their children are NOT walked
    - bodyPreview: first 80 chars, newlines replaced with spaces, "..." appended if truncated
    - hasBody: true if body is non-empty string after trim
    - renderTree returns null if rootId specified but not found
    - archiveCard(data, id): sets card.archived=true recursively on all descendants; returns {id, title, descendantCount} or null
    - unarchiveCard(data, id): sets card.archived=false recursively on all descendants; returns {id, title, descendantCount} or null
  </behavior>
  <action>
    1. **mongoose.cjs** -- Add four new functions:

       `countActiveDescendants(card)`: Recursively count children where `!child.archived`. For each active child, count += 1 + countActiveDescendants(child). Skips archived children and their subtrees.

       `renderTree(data, rootId, opts)`:
       - Parse opts: `depth` (default 1), `archiveFilter` (default 'active')
       - Compute `maxDepth`: if depth === 0, set to Infinity; else use depth value
       - Build breadcrumbs: if rootId, call getPath(data, rootId), return null if path is null. Breadcrumbs = path.slice(0, -1).map(c => ({id: c.id, title: c.title})). If no rootId, breadcrumbs = null.
       - Determine start cards: if rootId, startCards = findById(data, rootId); if no rootId, iterate data.cards
       - If rootId: push the root card as render entry at depth 0, then walk its children starting at depth 1
       - Archive filter function: 'active' -> !card.archived; 'archived-only' -> card.archived; 'include-archived' -> true
       - Walk function: for each card in array, if shouldInclude(card), create render entry {id, title, depth: currentDepth, descendantCount: countActiveDescendants(card), hasBody: !!(card.body?.trim()), bodyPreview: makePreview(card.body), created: card.created, archived: card.archived}. If currentDepth < maxDepth and card.children.length > 0, recurse into children at currentDepth + 1.
       - `makePreview(body)`: if !body return ''. Replace newlines with spaces, truncate at 80 chars, append '...' if truncated.
       - Return {breadcrumbs, cards: flatArray}

       `archiveCard(data, id)`: Find card by id (return null if not found). Recursive setArchived(card, true) on card and all descendants. Return {id: card.id, title: card.title, descendantCount: countDescendants(card)}.

       `unarchiveCard(data, id)`: Same as archiveCard but sets archived=false.

    2. **Update module.exports** to include: renderTree, archiveCard, unarchiveCard, countActiveDescendants

    3. **mongoose.test.cjs** -- Add test suites:

       `describe('countActiveDescendants')`:
       - Returns 0 for leaf card
       - Counts only active children (skip archived and their subtrees)
       - Counts all children when none archived

       `describe('renderTree')`:
       - Root view (no rootId): returns all root cards at depth 0, breadcrumbs null
       - Root view with depth 1: includes root cards and their direct children
       - Root view with depth 0: full tree, all cards at correct depths
       - Focused view: rootId specified, breadcrumbs show ancestors, root card at depth 0
       - Focused view depth 2: shows 2 levels of children
       - Default depth (no depth arg): card + direct children (depth 1)
       - Archive filter 'active': excludes archived cards
       - Archive filter 'archived-only': shows only archived
       - Archive filter 'include-archived': shows all
       - Render entry has correct fields: id, title, depth, descendantCount, hasBody, bodyPreview, created, archived
       - bodyPreview truncates at 80 chars with "..."
       - bodyPreview replaces newlines with spaces
       - hasBody false for empty/whitespace-only body
       - Returns null for nonexistent rootId
       - descendantCount uses active-only count

       `describe('archiveCard')`:
       - Archives card and all descendants
       - Returns {id, title, descendantCount}
       - Returns null for missing id

       `describe('unarchiveCard')`:
       - Unarchives card and all descendants
       - Returns {id, title, descendantCount}
       - Returns null for missing id

       Update sampleTree() to include some cards with body content for render tests. Add a helper `sampleTreeWithArchived()` that has some archived cards for filter tests.

    4. **Run tests**: `node --test .claude/burrow/test/mongoose.test.cjs`

       - Call `tree.unarchiveCard(data, id)`

         - `case 'children':` parse positional id, call handleGet(data, {id, depth: 1, archiveFilter: 'active'})
