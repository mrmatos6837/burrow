## Integration Check

**Result:** All cross-phase wiring verified. 0 broken connections.

### E2E Flows (4/4 complete)

1. **Fresh install** — `npx create-burrow` → `install.cjs` → `detect()` → `performInstall()` → file creation + CLAUDE.md sentinel
2. **Update** — `version.checkForUpdate()` → npm registry fetch → stderr notification → `/burrow:update` → `npx create-burrow --yes` → upgrade mode
3. **Passive notification** — `burrow-tools.cjs writeAndExit()` → `checkForUpdate(cwd)` → 24h cache check → conditional stderr notice
4. **Uninstall** — `install.cjs --uninstall` → confirmation → file removal → `removeSentinelBlock()` → empty dir cleanup

### Phase 13 Gap Resolution

Phase 13 resolved both integration gaps from the 2026-03-17 audit:
- **Gap 1 (checkForUpdate orphaned):** Now called directly from `burrow-tools.cjs writeAndExit()`
- **Gap 2 (writeBreadcrumbs missing from runRepair):** Entire `.source-dir`/breadcrumb mechanism removed — `/burrow:update` uses `npx` now

### Export/Import Map

All key exports consumed. 1 minor cosmetic note: `fetchLatestVersion` exported from `version.cjs` for testability but only called internally in production via `checkForUpdate()`.
