<task type="auto">
  <name>Task 1: Rewrite install.cjs CLI with readline, --yes, and --uninstall</name>
  <files>install.cjs</files>
  <action>
    Complete rewrite of install.cjs. Import engine functions from `.claude/burrow/lib/installer.cjs`. Structure:

    **Argument parsing** (process.argv):
    - `--yes` / `-y`: non-interactive mode, all defaults, no prompts
    - `--uninstall`: uninstall mode
    - Positional arg: target directory (default: process.cwd())
    - `--help`: print usage and exit

    **Main flow:**
    1. Parse args
    2. Resolve sourceDir (the burrow repo -- where install.cjs lives) and targetDir
    3. Run `detect(targetDir)` to determine mode
    4. Branch on mode:

    **Fresh install flow:**
    - If interactive: prompt for install path (default: cwd), prompt for CLAUDE.md opt-in (explain what it does, default yes)
    - If --yes: use defaults (cwd, CLAUDE.md yes)
    - Run performInstall(sourceDir, targetDir)
    - If CLAUDE.md opted in: writeSentinelBlock(claudeMdPath, BURROW_SNIPPET)
    - Print checklist output with ok()/skip() helpers
    - Print post-install "getting started" message

    **Upgrade flow:**
    - Print "Detected existing burrow installation"
    - Show version info if available: "Upgrading burrow: v{old} -> v{new}"
    - If interactive: ask confirmation (default yes). If --yes: proceed.
    - Run performUpgrade(sourceDir, targetDir)
    - Handle CLAUDE.md sentinel: replace block with new version
    - Handle legacy CLAUDE.md (has ## Burrow but no sentinels): wrap existing in sentinels or replace -- Claude's discretion
    - Print checklist output

    **Repair flow:**
    - Print "Detected partial burrow installation, {N} files missing"
    - List what's missing
    - If interactive: ask confirmation (default yes). If --yes: proceed.
    - Run performRepair(sourceDir, targetDir, missing)
    - Print checklist output

    **Uninstall flow (--uninstall):**
    - Print what will be removed (list dirs/files)
    - If interactive: require confirmation with default NO (destructive). If --yes: proceed.
    - Remove: .claude/burrow/, .claude/commands/burrow/, .claude/commands/burrow.md, .planning/burrow/
    - Run removeSentinelBlock on CLAUDE.md
    - Clean up empty parent dirs (if .claude/commands/ is now empty, remove it; if .claude/ is now empty, remove it; same for .planning/)
    - Print checklist of what was removed

    **Output helpers:** Keep ok(), skip(), fail() pattern from current installer. Add warn() for notices.

    **readline usage:** Use node:readline with createInterface({ input: process.stdin, output: process.stdout }). Create a simple `ask(question, defaultAnswer)` promise wrapper. Close interface when done.

    Zero external dependencies. CommonJS. Shebang: `#!/usr/bin/env node`.
  </action>
  <verify>
    <automated>node install.cjs --help</automated>
  </verify>
  <done>
    - install.cjs --help prints usage with flags described
    - install.cjs --yes runs fresh install non-interactively in a test directory
    - install.cjs --uninstall --yes removes all burrow files from a test directory
    - Interactive mode prompts work with readline
  </done>
