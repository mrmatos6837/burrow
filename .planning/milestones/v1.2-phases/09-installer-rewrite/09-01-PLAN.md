<task type="auto" tdd="true">
  <name>Task 1: Detection and sentinel engine</name>
  <files>.claude/burrow/lib/installer.cjs, test/installer.test.cjs</files>
  <behavior>
    Detection:
    - detect(targetDir) returns { mode: 'fresh' } when no burrow files exist at target
    - detect(targetDir) returns { mode: 'upgrade', version: string|null } when all core paths exist (.claude/burrow/burrow-tools.cjs, .claude/commands/burrow.md, .claude/commands/burrow/, .planning/burrow/cards.json)
    - detect(targetDir) returns { mode: 'repair', missing: string[] } when some core paths exist but not all
    - detect(targetDir) returns hasSentinel: true when CLAUDE.md contains <!-- burrow:start -->
    - detect(targetDir) returns hasLegacyClaude: true when CLAUDE.md contains "## Burrow" but no sentinel markers

    Sentinel CLAUDE.md:
    - writeSentinelBlock(claudeMdPath, content) appends sentinel-wrapped block to end of file, creating file if needed
    - writeSentinelBlock preserves existing CLAUDE.md content before/after sentinel block
    - writeSentinelBlock replaces existing sentinel block content on re-run (upgrade)
    - removeSentinelBlock(claudeMdPath) removes only the sentinel-marked section, preserving all other content
    - removeSentinelBlock is a no-op if no sentinel markers exist
    - Sentinel markers are: <!-- burrow:start --> and <!-- burrow:end -->
    - Line ending detection: match existing file (CRLF vs LF)

    Install/upgrade file operations:
    - performInstall(sourceDir, targetDir) copies .claude/burrow/, .claude/commands/burrow.md, .claude/commands/burrow/, creates .planning/burrow/ with empty cards.json, adds .gitignore entry
    - performUpgrade(sourceDir, targetDir) replaces .claude/burrow/ and .claude/commands/ but does NOT touch .planning/burrow/cards.json
    - performRepair(sourceDir, targetDir) copies only missing files (from detect's missing list), does not touch existing files
    - All operations return a results object with status per item (copied/skipped/preserved)
  </behavior>
  <action>
    TDD cycle. Write tests FIRST in test/installer.test.cjs using node:test, node:assert/strict, temp directories.

    Then implement .claude/burrow/lib/installer.cjs with these exports:
    - `detect(targetDir)` — scans for core burrow files, returns mode + details
    - `performInstall(sourceDir, targetDir, opts)` — fresh install file operations
    - `performUpgrade(sourceDir, targetDir, opts)` — upgrade with data preservation
    - `performRepair(sourceDir, targetDir, missingFiles)` — selective copy of missing files
    - `writeSentinelBlock(claudeMdPath, blockContent)` — insert/replace sentinel-wrapped content
    - `removeSentinelBlock(claudeMdPath)` — remove sentinel block only
    - `SENTINEL_START`, `SENTINEL_END` — exported constants for the markers

    Core paths to check in detection (relative to target):
    - `.claude/burrow/burrow-tools.cjs` (source engine)
    - `.claude/commands/burrow.md` (root command)
    - `.claude/commands/burrow/` (subcommands dir)
    - `.planning/burrow/cards.json` (data file)
